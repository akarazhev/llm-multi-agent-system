# Scripts Directory

This directory contains utility scripts for the LLM Multi-Agent System.

## Available Scripts

### `setup_env.sh` (Unix/Linux/macOS)

Sets up Python environment with dependencies.

```bash
./scripts/setup_env.sh
```

### `setup_langgraph.sh`

Sets up LangGraph integration.

```bash
./scripts/setup_langgraph.sh
```
